Here is some simple software, all released under the BSD license. If any of it proves useful, please let me know!

rps/rwho

These scripts query a set of remote machines to determine either what processes are running or who is logged in, respectively. The output of rwho looks like this:

   host         uptime                 users(idle)
comp1.graphic
comp2.graphic35 days,  8:20
serv.graphics21 days, 23:10       cdtwigg(34.00s) john(1:40m)

The output of rps looks like this:

 host             processes
comp1
comp2         josh:MATLAB (26.6%);
comp3         fred:MATLAB (25.7%);

The error handling in these could definitely be better, so if anyone fixes this please let me know and I'll update these versions. Download them here:

  • rwho
